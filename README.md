# Hostel Room Allocation Management System

## Overview

A Java-based system to allocate hostel rooms to students and manage room occupancy.

## Features

* Allocate room
* View allocated rooms
* Check vacant room
* Remove allocation

## Technologies Used

* Java
* OOP
* ArrayList
* Scanner

## How to Run

1. Open in Java IDE
2. Run `HostelRoomAllocation.java`
3. Use menu options
